# PMD的报告格式

摘要：PMD内置的报告格式介绍。

​ PMD可以生成多种格式的检测报告。有些格式的报告可以通过参数配置定制特性。执行或者配置错误可能导致有些问题未报告出来。不是所有格式的报告都显示所有信息。

​ 命令行的--format参数指定输出报告的格式，值可设置为sarif、codeclimate、csv、emacs、html、ideaj、json、summaryhtml、text、textcolor、textpad、vbhtml、xml、xslt、yahtml。如果某种报告格式支持参数配置，在命令行可以使用--property/-P进行设置。

🎗️只有在设置了--show-suppressed时，被压制的问题才会显示。

# CSV

​ 逗号分隔值形式。

​ 这种格式只有违规问题，被抑制的问题和错误将被忽略。

"Problem","Package","File","Priority","Line","Description","Rule set","Rule"
"1","net.sourceforge.pmd","/home/pmd/source/pmd-core/src/main/java/net/sourceforge/pmd/RuleContext.java","2","124","Logger calls should be surrounded by log level guards.","Best Practices","GuardLogStatement"
"1","net.sourceforge.pmd.benchmark","/home/pmd/source/pmd-core/src/main/java/net/sourceforge/pmd/benchmark/Benchmarker.java","3","58","This for loop can be replaced by a foreach loop","Best Practices","ForLoopCanBeForeach"

​ 配置参数:

  • problem: 是否包含problem信息列. 默认: true.
  • package: 是否包含package信息列. 默认: true.
  • file:是否包含file信息列. 默认: true.
  • priority: 是否包含priority信息列. 默认: true.
  • line: 是否包含line信息列. 默认: true.
  • desc: 是否包含desc信息列. 默认: true.
  • ruleSet:是否包含ruleSet信息列. 默认: true.
  • rule:是否包含rule信息列. 默认: true.

# ideaj

​ IntelliJ IDEA继承PMD时的输出格式。

​ ⚠️这种格式只能在IDEA中使用。

​ 两种调用方式：

​ 单文件调用

pmd check -d src/Foo.java -R rulesets/java/quickstart.xml -f ideaj -P fileName=src/Foo.java -P sourcePath=/home/pmd/src -P classAndMethodName=Foo

​ 目录检测

pmd check -d src -R rulesets/java/quickstart.xml -f ideaj -P sourcePath=/home/pmd/src -P classAndMethodName=.method

​ 报告例子

Logger calls should be surrounded by log level guards.
 at Foo(:124)
This for loop can be replaced by a foreach loop
 at Foo(:58)

​ 配置参数

  • classAndMethodName: Class and method name, pass .method when processing a directory.
  • sourcePath:
  • fileName:

# json

​ josn格式，一个单独的json对象，包含一些header信息，根据文件分组的问题信息。有以下几个一级节点：

  • formatVersion: an integer which will be incremented if we change the serialization format
  • pmdVersion: the version of PMD that produced the report
  • timestamp: explicit
  • files: an array of objects (see the example)

# xslt

​ 应用XSL转换的XML。

​ PMD提供了一个默认的内置样式表，可以通过xsltFilename参数配置自定义样式表。可以在默认样式表 pmd-nicerhtml.xslopen in new window 的基础上进行定制修改。

​ 网上的一些样式表: https://github.com/pmd/pmd/tree/master/pmd-core/etc/xslt.

​ 报告例子:

  • Example with pmd-nicerhtml.xslopen in new window
  • Example with html-report-v2.xsltopen in new window - includes charts. It requires javascript enabled and uses jQueryopen in new window, DataTablesopen in new window, and Vegaopen in new window for charting.

可配置参数:

  • encoding: XML 编码格式, 默认 UTF-8.
  • xsltFilename: XSLT文件名.
